# SOS Emergency React Native

A full-stack emergency safety application built using React Native, React.js, Node.js, and modern web technologies. The application helps users quickly send SOS alerts, access emergency contacts, view safety tips, and improve personal safety during emergencies.

---

## Features

* Emergency SOS trigger system
* Quick emergency contact access
* Real-time alert handling
* Safety tips and awareness section
* Responsive frontend interface
* Backend API integration
* Mobile application support
* Cross-platform support

---

## Tech Stack

### Frontend

* React.js
* Vite
* CSS
* Axios

### Mobile

* React Native

## Installation

### 1. Clone the Repository

```bash
git clone https://github.com/madhulatanishad123/SOS-Emergency-React-Native.git
```

### 2. Navigate to Project Folder

```bash
cd SOS-Emergency-React-Native
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```

Frontend will run on:

```bash
http://localhost:5173
```

---

## Backend Setup

```bash
cd backend
npm install
npm start
```

Backend server will run on:

```bash
http://localhost:5000
```

---

## Mobile Setup

```bash
cd mobile
npm install
npx react-native run-android
```
